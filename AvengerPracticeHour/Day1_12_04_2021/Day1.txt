              1. Type Casting

                a.Implicit Casting
                  -Done by compiler
                  -Smaller to bigger data type
                  -No loss data

                b.Explicit Casting

                  -Done by developer
                  -Bigger to smaller
                  -we can loss data

                  double a=5.3;
                  int b=(int)a;

                  double a=5;
                  int b=(int)a;
